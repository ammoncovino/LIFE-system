## System Architecture (4 Layers)

```
LAYER 1 — MASTER TOTEM (Internal)
  Full biological truth model. 8 sections per species.
  Used by: AI, leadership, documentation.

LAYER 2 — STUDENT TOTEM (Public Display)
  K-5 level. 8-question simplified model. Physically printed.

LAYER 3 — DIGITAL TOTEM (Interactive AI)
  QR → Quick Learn → Go Deeper → Ask About This Animal
  AI governed by CEI constraints.

LAYER 4 — NARRATIVE SUITES (Interpretive)
  20 rooms. Symbol-based learning. No characters/cartoons.
  Perception → Responsibility → Illusion → Signal → Naming → Expansion → Energy → Synthesis
```
