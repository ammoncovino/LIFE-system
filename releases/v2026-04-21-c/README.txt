Design
------
  - Full-page parchment body
  - Dark wood title band:   common name (Cinzel Bold) + scientific name (Cormorant Italic)
  - Left column:            8 locked questions + K-5 answers (10-19 words each)
  - Right column:           AI-generated painterly illustration (life-like, species-accurate)
  - Wood band:              "Think About This" per-species reflection question
  - Footer wood band:       "What a living thing can sense becomes its reality."
                            "Environment shapes design."

Locked Spine (unchanged from prior releases)
--------------------------------------------
  1. What can it see, hear, or feel?
  2. Where does it live — and why?
  3. How does it send messages?
  4. What does it do when something changes?
  5. How does it learn?
  6. How does it have babies?
  7. What are its limits?
  8. What does this teach us?

Rules preserved
---------------
  - WE SIMPLIFY ANSWERS — NOT STRUCTURE
  - K-5: 10-19 words per answer, forbidden-words list enforced
  - All 18 species use the same 8 questions in the same order
  - Footer (both lines) locked verbatim
