# species_specific_totems_to_print/

The print bundle. Open an animal folder, send to printer.

These are the **styled totems** — the ones with photos. One file per species, each ready to print.

When new species land in `by_species/`, their styled totem gets copied here so this folder always reflects what's print-ready.

Source of truth lives in `by_asset_type/styled_totems/` and `by_species/<animal>/`. This folder is the print queue, not a source.
