# Employee Reading

Everything an employee might need to read. If you are staff — start here.

This is the only folder you have to know about. Open it, find what you need, go to work.

## What's in each folder

### `feeding/`
Everything about feeding the animals — the manual, the wall poster, the diet verification sign, the addendums, and zero-waste.

- `GOLD_STANDARD_DIET_RULE.md` — match the bowl, don't interpret. Sulcata is the only hay-only species.
- `SM_01_Feeding_Complete.pdf` — complete feeding manual
- `QR_02_Feeding_Protocol.pdf` — wall-mounted feeding protocol
- `SIGN_Diet_Verification.pdf` — diet verification sign (posted at every bowl)
- `part07_feeding_addendums.md`, `part08_diet_verification.md` — markdown source
- `ZERO_WASTE_FOOD_LOOP.md`, `QR_04_Zero_Waste.pdf`, `part06_zero_waste.md` — zero-waste

Diet cards by species: `../by_species/<species>/diet_card_<species>.pdf`
