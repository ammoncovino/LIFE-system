## What you'll find in each folder

A complete species folder contains:
- `<species>.md` — canonical species file (wild diet, range, scientific name)
- `diet_card_<species>.pdf` — gold-standard diet card (printable)
- `door_totem_<species>_<site>.pdf` — encounter entry door sign (Houston / San Antonio)
- `walkby_totem_<species>.pdf` — corridor pull-tab totem
- `micro_totem_<species>.pdf` — small handheld card
- `styled_totem_<species>.pdf` — full glamour totem
- `student_totem_<species>.pdf` — STEM/classroom version

Some species don't yet have all 7 — that's the remaining inventory work. The 4 tortoises (red-foot, yellow-foot, russian, leopard) are NEW in v2026-04-28d and have a complete set.
